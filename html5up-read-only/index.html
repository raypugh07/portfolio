A Few Accomplishments

Below you will find a few projects demonstrating my skills and experience.
If you are interested in viewing more projects/code I have written please check out my Github.

Meet Gibber!

  • Gibber is a social blogging single page app.
  • On Gibber you can create, comment on and like posts without the clutter of typical social media.
  • I built Gibber using React on the Frontend and Node Express on the Backend.
  • The database is managed on Postgres and I implemented auth0 to handle the authentication.

Empowered Conversation

  • Empowered Conversation is a survivor activated tool to have difficult conversations with family and friends.
  • This app is a part of a case study at the University of Michigan.
  • I was a Full Stack Developer for this project using React.JS on the front end and Java Spring Boot for the backend.

Java Backend

  • Built using Java Spring Boot.
  • REST API server.
  • PostgreSQL database.
  • Oauth2 Authentication.
  • Deployed to Heroku.

The Confessional

  • A single page web app to seek forgiveness.
  • All confessions are converted to null before hitting database for privacy.
  • Frontend built using React.
  • Styling was done with React Semantic UI library.
  • Backend built using Java Spring Boot.
  • PostgreSQL database.
